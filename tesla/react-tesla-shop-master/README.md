# 前台目录

├─api // 网路请求代码、工具类函数和相关配置<br>
├─assets // 字体配置及全局样式<br>
├─baseUI // 基础 UI 轮子<br>
├─components // 可复用的 UI 组件<br>
├─routes // 路由配置文件<br>
└─store // Redux 相关文件<br>
App.jsx // 根组件<br>
main.jsx // 入口文件<br>

# 服务器

1.  后端 server 采用 Koa 搭建

# 完成功能

1. 发现页 (swiper、detailed )

3. 地图（百度 api）

5. ......

# 启动

1. 在 server 中：

```
npm i
node index.js
```

2. 在 src 中：

```
yarn
yarn dev
```
